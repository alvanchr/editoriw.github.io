ini adalah contoh elemen font, dimana terdapat beberapa property didalam font seperti

  • font family : digunakan untuk mengubah jenis font
  • font size : digunakan untuk mengubah ukuran font
  • font weight : digunakan untuk mengatur ketebalan font
  • font variant : digunakan untuk mengubah font menjadi small caps
  • font style : digunakan untuk mengubah font menjadi cetak miring
  • line height : digunakan untuk mengatur spasi antar baris
